**Инструкция по установке:**

1. Установите зависимости:
```bash
npm install express cors
```

2. Запустите сервер:
```bash
node server.js
```

3. Откройте в браузере страницу пользователя:
```
http://localhost:3000
```

4. Запустите сервер:
```bash
node admin.js
```

5. Откройте в браузере страницу пользователя:
```
http://localhost:8080/admin
```
